# 项目介绍

    本项目是基于Github开源项目BasicMod所构建的《杀戮尖塔》的游戏模组。

    使用美术素材均来自游戏本体及ChatGPT4o生成。
  
# 使用说明

     下载zmdzzz.jar文件，放入D:\SteamLibrary\steamapps\common\SlayTheSpire\mods路径（请调整至你的mods路径位置），从Steam中打开游戏即可使用
